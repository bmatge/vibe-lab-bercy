## Valeur pour la communication

- **Gouvernance des sites** — Ecosysteme surveille tous les sites ministeriels : performance, accessibilite, SSL, DSFR. Problemes d'image detectes avant les usagers.
- **Connaissance du portail** — Portail Eco Browser cartographie tout le contenu, croise les analytics, identifie les pages obsoletes ou redondantes.
- **Coherence de marque** — ChartsBuilder, LimeSurvey DSFR, organigrammes : l'identite visuelle de l'Etat portee sur chaque point de contact.
- **Contenus engageants** — Comparateurs, dataviz, cartes interactives : des contenus a forte valeur pour le portail et les citoyens.

---

## Accessibilite : un chantier taille pour l'IA

- **Audit automatise permanent** — Scan RGAA de tous les sites ministeriels en continu, pas tous les 3 ans
- **Correctifs generes par l'IA** — Chaque non-conformite accompagnee du code correctif precis (ARIA, alt, HTML)
- **Controle integre pour les contributeurs** — LeFouineur verifie l'accessibilite en temps reel pendant la redaction
- **Formation par l'action** — Chaque recommandation est une micro-formation : on apprend en corrigeant

> La mission passe de prescripteur a facilitateur : elle ne dit plus seulement « corrigez », elle fournit les outils pour corriger.

---

## Securite et souverainete : risque nul

> Code public · Donnees publiques · Usages publics

Aucune donnee personnelle, aucun SI sensible, aucun secret.

- **Open source** — Tout le code publie sur GitHub — visibilite et reutilisabilite interministerielle
- **API publiques exclusivement** — data.gouv.fr, INSEE, Annuaire du service public, data.economie.gouv.fr
- **IA souveraine integree** — Albert (Etalab) utilise dans Portail Eco Browser et LimeSurvey
- **Deploiement cloud sans risque** — Conteneurs Docker portables — aucune donnee sensible hebergee

## La demande

- **1,5 a 3 ETP dedies** — Profils tech/produit qui connaissent les metiers du ministere. La premiere annee, le Lab peut fonctionner avec 1,5 ETP en mobilisant deux ou trois agents deja en poste qui consacreront d'un tiers a deux tiers de leur temps au sujet.
- **~1 000 €/mois de licences** — Claude + GitHub Copilot (~350 €/personne/mois)
- **0 € d'infrastructure** — GitHub, CI/CD, Docker : tout est deja en place

**Cout annuel licences : ~15 600 €** — Soit 2,4 % du cout prestataire deja evite

> Le Lab s'autofinance des son premier mois.
